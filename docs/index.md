## 程序的基本构成：

在学习任何一门编程语言的时候，我们首先要了解一门编程语言的基本组成。它包括：
- 数据类型：Python 中的数据类型有 float, int, boolean, char, list, dict, set 等。
- 操作符：对于float 和 int 数据类型有加`+`、减`-`、乘`*`、除`/`等常见的操作符。
- 函数：有内置函数和自定义函数。
- if 分支语句：当语句是真的时候，运行代码分支 1，当语句结果是 False 的时候，运行另一个代码分支。
- 循环语句: while/for loop。
- OOP: object-oriented programming, 即当很多变量和函数与一个主体相关的时候，我们最好定义这个主体，并把相关变量和函数绑定到这个主体上，方便之后访问。
- module：一个或者多个脚本构成，当程序代码比较大的时候，所有程序放在一个脚本中
